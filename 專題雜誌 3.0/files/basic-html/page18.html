                                    佑賢の心聲第一次見面是集體共畫，那時他感覺跟他一組的樂齡同學特別熱情，破冰活動中樂齡同學有兩位，其中一位就是忠宏的夥伴，大家都叫她大孫姐，她會主動出來跟大學生們討論要如何畫以及問他們的想法，他們這組氣氛特別好，冷場？絕對是不存在的。大孫姐的另一位大學生夥伴是佑賢，最初的他害怕自己缺乏跟陌生人相處的能力，相見的第一天他很內向不太敢跟樂齡溝通，也不敢分享自己的想法，默默觀察是他當下的代名詞，但經過樂齡同學開朗的個性以及更多的交流經驗過後，他開始會跟樂齡一起討論自己創作的理念。佑賢與大孫姐的初次見面原本忠宏很擔心有代溝的問題，但後來他發現似乎是因為相處時間太短以及不知該如何開啟話題的原因，導致沒有交流也就察覺不到是否有代溝，像他這種慢熟的人需要時間也需要勇氣。不擅言詞的我...也可以嗎？大孫姐（左一）和忠宏（右三）在破冰活動的合照佑賢:其實也沒那麼可怕嘛，下次可以主動一點。16
