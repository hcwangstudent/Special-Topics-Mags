                                    昱臻與美珍姐在成果發表日的合照在過去，昱臻心裡一直存在著對於老人家的刻板印象，可他還是選擇了來嘗試、來親身感受，希望訓練自己與不同年齡層的人溝通。而美珍姐的出現恰巧讓他感受到樂齡的好，讓他消除了對於長輩們的固有想法。這樣的跨世代交流對昱臻是有意義且值得深思的。在這裡，他學到了要去認識不同年齡層的人，不只是與同年紀的人處在一起，偶爾也要與各個年齡層的人交流互動，去發現他人擁有而自己所欠缺的。在未來，昱臻會更有耐心的處理工作，應對不同個性的人也會有特定的溝通方法。而在身為打工人的昱臻眼裡，職場上的跨世代相處比課堂更難，也更不容易親近，職場上會有強烈的競爭力，可能會有同事為了業績壓力而只在乎自己，不願意互相幫助。如人飲水，冷暖自知課堂接近尾聲，政良也改變了對於樂齡同學的看法，如果說熱情與創造力是年輕人的代名詞，那麼在這裡，這也可以是對樂齡同學們的代名詞。他說「我發現樂齡同學依舊很願意學習、嘗試新的事物，像是粉彩畫或新科技等，並且他們的創造力依舊豐富、不死板，我甚至覺得都比我好了。」昱臻:我感覺唯有自己體驗到才知道其中的意義，過程中自己有改變、有收穫，樂齡同學與我們沒有什麼大不同，反而有更多累積而來的經驗。23
