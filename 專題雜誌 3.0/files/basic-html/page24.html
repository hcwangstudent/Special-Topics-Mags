                                    菀庭與美珍姐在成果發表日的合照－當磨合轉化為理解，共學共融既是社會責任也是職前準備－蛻變而收穫不只有社交，菀庭在美珍姐身上看到了＂嘗試＂的重要性，很多人都有好奇心，卻不一定有行動，而美珍姐願意去樂齡大學學習、接觸不會的畫畫，更願意跟陌生的年輕人交流，這是菀庭所欠缺的，可這樣的人是她很佩服的。菀庭起初以體驗者自居，但社會實踐為銀青共學注入了意義。電子雜誌的實作轉化為一種使命感，縱使讀者人數未知，她仍期盼能發揮影響力，減少世代衝突。面對長輩，菀庭從來沒有過多的偏見，不管甚麼年齡總會遇到契合與不契合的人，這次的相處更多的是為她建立了一個跟跨世代共融的勇氣。從一開始的緊張到最後的歡聲笑語，對於菀庭是一次小試牛刀，一次進入職場前的適應期，她未來會以更客觀的心態去面對職場人。她選這門課的初衷就是實踐，想累積一點職場技能，她說：「編輯雜誌是很接近職場的技能，所以我想來學習，過程中了解其中的不易與要具備的核心，也意外提前感受到要是未來需要跟年齡差大的長輩交流該如何？這些經驗碎片都會被我收在我的行囊中，相信日後它們會派上用場！」你的身上也有閃光點能力從不是一蹴而就菀庭:第一次跟樂齡族相識就能留下一個圓滿的體驗感，以後我能更加自在地去認識那些＂老＂朋友。22
