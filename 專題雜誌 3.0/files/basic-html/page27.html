                                    少英與黃直在成果發表日的合照心妤與黃直在成果發表日的合照少英：在課堂上與樂齡同學的互動中會找出共同的話題來交流拉近彼此的距離。同樣在職場上也是可以利用大家共同感興趣的話題或是共同解決的問題、做的項目、打拼的過程來拉進與同事之間的距離，這都是很相似的經驗。這次共畫少英所學到的不僅僅是粉彩畫技巧，更多的是通過對樂齡同學的交流讓自己對情緒處理有更高的能力，也同時很敬佩樂齡同學對人生的樂觀態度和處理情緒的智慧。黃直大哥帶給她的就是永遠不用去對還沒有發生的事情預設立場，你所擔心的並沒有你想像的那麼糟，用樂觀的態度來看待未知的世界。未來在職場上會遇到更多形形色色的人，在這堂課程上就像一個實習課，讓自己先預習一下遇到陌生的、不同年齡層的人，要如何與之相處，而作為長輩的經驗之談，職場利益的牽扯跟上課肯定是不同的，但有相似之處。就像模擬考試一樣，儘管練習了無數次，真正大考時也沒辦法保證交出滿分的答卷，但起碼知道會面對哪些題目，能夠讓自己接近理想的結果。練習是為了好的結果心妤從與黃直大哥的交流中學會放慢步調，去傾聽和觀察，這對她來說是一種價值，因為平常生活節奏快，很少真正靜下來感受人與人之間的美好。所以跨世代共處不只是互相陪伴，更是一種學習。年輕人可以吸收長輩的人生經驗，而長輩也能感受到新的想法和活力，這是一種雙向交流。彼此學習，互相成長心妤:以前我覺得長輩可能比較嚴肅，但參與後發現他們其實也有很多創意和童心，這讓我不再只把他們看作『上一輩人』，而是可以平等互動的夥伴。25
