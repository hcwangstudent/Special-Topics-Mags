                                    1.認真傾聽尊重對方2.學會看時機3.確認對方與自己在同一件事情上4.避免指責、針對性詞語5.注意情緒語調、身體語言溝通的核心有哪些？Joyce在聊到恐懼這個主題時，Joyce能透過政良清楚的比喻來很好的了解他心中的想法，Joyce說：「政良提及人們最在乎的是哪個部分，恐懼就會在那出現。他還有分類喔！那我們兩個都屬於完美主義者，他的完美主義者是說，他覺得他有很多潛力，可以做得更好，比如說可以考到兩百分，可是為什麼現在只有一百分。就是我能超過我的既有水準，但是我為什麼一直沒有辦法達到，這是政良心裡的恐懼。」當他們第一次在破冰活動相遇並共畫的時候，Joyce不知道該做甚麼，所以她會問政良怎麼畫呀？問了很多問題，然後大學生也會耐心的講解，在這樣一個互動下，政良也會問Joyce想畫什麼，在接下來的相處中，Joyce發現她的學伴是越陳越香的那種性格，她表示：「政良他不是一個很愛現的人，但是他很願意表達他的觀感，除了畫作之外，我們還聊了很多生活上的、私人的思緒。」在Joyce心裡，比起視線相對，或許真誠的對話更為重要，政良會認真回應Joyce的問題，傾聽她的想法並迅速了解到她想表達的重點。關於溝通表達力，如果說從政良的角度來看，Joyce認為他學這門課可以掌握到如何觀察情緒，去理解他的同事或長官的情緒，然後他可以理出這個情緒的癥結點在哪裡，以便跟客戶或廠商在溝通、做交易的時候可以做出合適的回應，所以溝通和傾聽在工作上是很關鍵的。Joyce:政良不是一個很多話的人，但是他都講重點，是一個願意溝通的人。資料來源：掌握「溝通技巧」5大核心，增進人際關係必學的職場溝通術！Joyce:他會把我聊天中跟他提到的內容加到電子雜誌中，像是我跟他提到代贖這個詞，就是基督教代替我們贖罪，我在講的時候，只是簡單帶過，但是他卻在雜誌內容下面有描述詞的定義，證明他是真的有認真傾聽並記住我說的話。31
